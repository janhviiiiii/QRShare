• The "QR Share" project stands out for its comprehensive approach, combining QR code generation, scanning, and various data transfer options into a single, integrated platform.
• This holistic design addresses the need for a streamlined and versatile system that enables users to create, scan, and share QR codes effortlessly

• __Description:__  Developed an Android application enabling users to generate, scan, and share QR codes for various data types such as text, URLs, contact details, email, event, socials etc.
• __Technologies Used:__ Java, XML, SQLite, Android Studio.
• __Key Features:__ QR Code Generation and Scanning, Favorite and History feature, Data Storage and Transfer, User Authentication, UI/UX Design 
• __ __Achievements:__ __
    o Developed a robust application facilitating seamless data transfer.
    o Enhanced understanding of Android development, improved skills in Java and SQLite.
